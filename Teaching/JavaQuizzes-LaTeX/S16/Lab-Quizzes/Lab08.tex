\Instructions

\begin{enumerate}
\vspace{2em}

\item \pts{2} \tf A concrete class implementing an interface must implement all the methods defined in the interface.
\ifdraft
\Ans True. 
\fi
\vspace{3em}
\item \pts{2} \tf An interface can have private methods.
\ifdraft	 
\Ans False.
\fi
\vspace{2em}

\item \pts{2} \tf A class can implement multiple interfaces.
\ifdraft
\Ans True. The result will be the union of all the capabilities of all of the interfaces.
\fi
\vspace{2em}

\item \pts{2} \tf A field can be declared but not initialized in an interface.
\ifdraft
\Ans False. Fields that are declared in an interface  \textbf{must} also be initialized in the interface.
\fi
\vspace{2em}

\item \pts{2} \tf The following interface is correctly declared.
\begin{lstlisting}
public interface System
{
   String getSystemVersion();
   String getDefaultUser();

   default String getSystemInfo()
   {
      return "User: " + getDefaultUser() + ", Version: " + getSystemVersion();
   }
}
\end{lstlisting}
\ifdraft
\Ans True. Methods with the default keyword can contain non-empty bodies.
\fi
\end{enumerate}
